설치방법

  1. vcpkg 설치(참조)
  2.  .\vcpkg install boost boost:x64-windows  실행.

    3. 자동으로 설치됨. 설치 위치는 ../vcpkg/installed/

사용방법

  1. Configuration properties > C/C++ > General > Additional Include Directories 에 ../vcpkg/installed/boost include dir 추가 : header file include path 추가
  2. Configuration properties > Linker > General > Additional Library dependency 에
    3. //rev 데스크탑에서 VS 설정 확인하기
  3. boost 라이브러리의 빌드된 dll 파일을 프로젝트 디렉터리의 루트에 복사
    4. //rev 데스크탑에서 VS 설정 확인하기
vcpkg에서 user-wide integration 설치했을 시 사용방법에 언급된 별도의 설정과정 없이 모든 프로젝트에서 바로 boost 사용 가능.
