vcpkg

Open source library를 쉽게 사용할 수 있도록 해주는 open source build tool. Visual studio extension.

필요 Spec

  • Visual Studio 2015 이상
  • Windows 7 이상
  • Powershell

설치방법

  1. Clone from GitHub: git clone https://github.com/Microsoft/vcpkg
  2. Run the bootstrap script in the vcpkg folder: powershell -exec bypass scripts\bootstrap.ps1
  3. Now vcpkg.exe is in the root of the repository
  4.  ./vcpkg integrate install  실행

    5. user wide integration. vcpkg를 통해 설치한 라이브러리를 모든 visual studio 프로젝트에서 별도의 include, linking 작업 없이 사용할 수 있게 해줌.

사용법

  • vcpkg를 통해 설치한(관리하는) open source library 목록 출력 :  ./vcpkg list 
  • vcpkg 도움말 출력 :  ./vcpkg help
