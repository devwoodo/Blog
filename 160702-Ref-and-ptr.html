title: 언제 reference type을 써야하고 언제 pointer type을 써야할까? tag:
  1. C++에서 Pointer 사용은 가능한 최대한 피할 것.

  2. Reference type은 가르키는 object를 정의하면 이후 다른 object를 가르키도록 변경할 수 없음.

  3. Reference type에는 아무 object도 가르키지 않는 null을 줄 수 없음.

  4. Pointer type이건 reference type 이건 return 시에는 reference type으로.(이거 타당성?)
